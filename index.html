Kaliteli bir site nasıl kodlanır?

Merhaba! eyer kaliteli web sitesi nasıl yapıldığını öğrenmek için geldiysen doğru yerdesin, sana göstereceğimiz adımları uygularsan başırılı bir web sitesi yapabilirsin

eyer kaliteli bir web sitesi oluşturmak istiyorsak 3 adet kodlama diline ihtiyacımız var.

1.HTML

2.CSS

3.JavaScript

HTML Nedir?

HTML, bir işaretleme dilidir. Bütün siteler HTML dilini kullanmak zorunda. Çünkü, HTML siz bir web sayfası oluşturulamaz bu yüzden HTML, site yaparken çok önemli bir işaretleme dilidir

CSS Nedir?

CSS, te bir işaretleme dilidir. CSS, sayfamızın tasarımı için oldukça önemlidir.Örneğin bir buton oluşturduk ve o butonun arka planın büyüklüğünü vs için CSS kullanırız. Unutmayın kaliteli bir web sitesi için tasarım şart.

JavaScript Nedir?

JavaScript bir programlama dilidir. JavaScript ile sitemizin backend(arka plan) kısmını ayarlayabiliriz ve JavaScript ile sitemmize birsürü özellik ekleyebiliriz. Örneğin sitenin en üstü kısmında olan arka plan butonuda JavaScript ve CSS ile yapıldı
